# Sales-Analytics-Dashboard-PowerBI
Interactive Sales &amp; Profit Dashboard built using Power BI

An interactive dashboard built to analyze sales performance, profit, margin %, category trends, regional sales, and channel comparison for business decision-making.

Tools & Techniques Used

Power BI (DAX Measures, Data Modeling, Visualization)

Power Query (Data Cleaning, Transformation)

CSV Dataset Processing

Key Insights Visualized

Overall KPIs: Total Sales, Total Profit, Profit Margin %, Average Order Value, Units Sold

Monthly Sales and Profit Trend

Category-wise Sales and Profit Comparison

Region-wise Performance Analysis

Sales Channel Comparison (Online vs Offline)

Dashboard Objectives

Identify high-performing categories and regions

Compare profitability across channels

Track month-wise sales and profit progression
